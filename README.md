## 📌 Описание проекта
Данный проект представляет собой комплексное продуктовое исследование, совмещающее в себе **статистический анализ результатов A/B-тестирования** и **моделирование Unit-экономики** для e-commerce платформы. 

В условиях изменения продуктовых фич или маркетинговых механик критически важно оценивать не только статистическую значимость изменений в поведении пользователей, но и их прямое финансовое влияние на бизнес-модель компании.

**Цель проекта** — провести валидацию результатов A/B-теста, оценить сдвиги в ключевых продуктовых метриках и рассчитать схождение Unit-экономики для принятия решения о раскатке новой фичи на всю аудиторию.

## ⚙️ Основные этапы и логика исследования

### 1. Проверка дизайна и качества сплитования (A/A и A/B валидация)
* Анализ размеров выборки в контрольной (A) и тестовой (B) группах.
* Проверка на отсутствие пересечений пользователей между сегментами (data sanity check).
* Оценка распределения трафика во времени для исключения внешних факторов и багов сплит-системы.

### 2. Статистический анализ метрик (A/B Тестирование)
В зависимости от природы метрик были применены соответствующие статистические критерии:
* **Конверсия (CR):** Оценка качественных изменений долей с помощью критерия Хи-квадрат ($\chi^2$) Пирсона или Z-теста.
* **Средний чек (AOV) и Выручка на пользователя (ARPU):** Проверка непрерывных метрик. Из-за скошенности и наличия выбросов, характерных для e-commerce покупок, применялись T-критерий Стьюдента и непараметрический критерий Манна-Уитни.

### 3. Расчет Unit-экономики (`df_unit`)
Результаты статистического анализа были интегрированы в финансовую модель на уровне пользователя/заказа:
* Расчет метрик **User Acquisition, CAC, Revenue, Margin, Fixed/Variable Costs**.
* Формирование итоговой витрины данных `df_unit` для оценки маржинальной прибыли проекта в разрезе тестовых групп и когорт.

## 📊 Дашборд Power BI
Для продуктовой команды и менеджмента был разработан интерактивный дашборд, агрегирующий результаты эксперимента:

1. **A/B Test Dashboard:** Динамика метрик по группам А и В, доверительные интервалы, значения $p$-value для проверяемых гипотез (визуальный маркер успешности теста).
2. **Unit Economics Calculator:** Карта метрик финансового здоровья продукта, наглядно показывающая, как изменения в конверсии или чеке масштабируют общую прибыль компании и окупаемость затрат.
